🚀 PrimeNexus LXC/LXD Ultra Installer

PrimeNexus ek next-generation, fully automated LXC + LXD installer hai jo Ubuntu aur Debian systems ke liye design kiya gaya hai.
Ye script ek complete container deployment ecosystem provide karta hai — ek hi command me install, configure aur deploy.

---

✨ Features

- ⚡ One-click LXC + LXD installation
- 🧠 Smart system detection (RAM, CPU, Disk)
- 🎛️ Interactive CLI menu system
- 📦 Auto container deployment
- 🌐 Network bridge auto configuration
- 🔐 Security hardening (Firewall, Fail2Ban, SSH protection)
- ⚙️ Performance optimization (CPU, RAM, Kernel tuning)
- 📊 System validation & report generation
- 🧾 Advanced logging system
- 🎨 Premium terminal UI

---

🖥️ Supported Systems

- Ubuntu (18.04+)
- Debian (10+)

---

📥 Installation

Run this command to install:

bash <(curl -s https://raw.githubusercontent.com/YOUR-USERNAME/lxc-installer/main/lxc-installer.sh)

---

⚙️ Usage

After running the script, you will see an interactive menu:

- Full Install (LXC + LXD + Container)
- Install only LXC/LXD
- Deploy container only
- System tools & diagnostics

---

🚀 Quick Commands

lxc list
lxc launch ubuntu:24.04 my-container
lxc exec my-container bash

🔐 Security

PrimeNexus automatically configures:
